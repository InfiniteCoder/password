Is My  Safe?

This website checks if your password has appeared before anywhere. "Anywhere" generally refers to password hacks and leaks. If the count is anything other than 0, you shouldn't use that password, as hackers will most likely try to hack your account using such publicly available passwords.

Website created by Aditya Patil. You can visit my blog at infinitecoder.github.io if you'd like.

API for checking passwords provided by Troy Hunt. If you found this website useful, do consider sending some donations to Troy here.
